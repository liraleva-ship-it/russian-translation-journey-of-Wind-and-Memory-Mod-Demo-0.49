# Russian Translation & QoL Improvement Mod 

> **Версия игры:** Demo **v0.49** — перевод и правки сделаны под эту сборку.

Масштабная руссификация и модификация, полностью перерабатывающая игровой опыт, добавляющая множество удобных функций для игроков.

## 🌟 Основные особенности

* **Полный перевод и Uncensored:** Качественный и аутентичный перевод на русский язык с абсолютным снятием любых цензурных ограничений оригинальной версии.
* **QoL (Quality of Life) Улучшения:** Множество исправлений интерфейса, навигации и игрового процесса, делающих прохождение гораздо более комфортным и плавным.
* **Геймплейные нововведения:** Точечные изменения игровых механик, исправления баланса и скрытых багов для улучшения общей динамики игры.

## 🛠️ Установка

1. Скачайте архив с последней версией модификации из раздела **Releases** (или соберите из исходников репозитория).
2. Распакуйте содержимое архива в корневую директорию установленной игры с заменой файлов (при необходимости).
3. *Перед установкой рекомендуется сделать резервную копию папки `Data`.*
