# Перевод (`.trans` проекты Translator++)

Здесь лежат исходные проекты перевода `rvdata2`-файлов игры в формате
**Translator++** (`.trans`). Сами `rvdata2` в репозиторий не входят.

> Игра: Demo **v0.49**.

## Формат и хранение

- Файлы `.trans` — большой минифицированный JSON (5 колонок: `Original Text` /
  `Initial` / `Machine translation` / `Better translation` / `Best translation`).
- Хранятся через **git-lfs** (см. корневой `.gitattributes`). Коммитим в
  **родном минифицированном виде** — тогда импорт обратно в Translator++ идёт 1:1.
- GitHub такие файлы в вебе не рендерит и diff не покажет (слишком большие) —
  это ожидаемо. Ревью/правки ведём в Translator++, git используем как
  версионируемый бэкап и общий источник правды.
- Кому нужен читаемый локальный diff — beautify JSON у себя на лету (в репозиторий
  разворачивать не нужно).

## Кто что переводит

Файлы у переводчиков **непересекающиеся** — держим их раздельно, поэтому
мерж-конфликтов нет. Финальный импорт собирается из всех частей.

| Файл | Переводчик | Охват (`rvdata2`) |
|------|------------|-------------------|
| `supplementary.trans` | melstroy | Classes, Items, Weapons, Skills, States, System, … |
| _(добавить)_ | _ник 2_ | _его файлы_ |
| _(добавить)_ | _ник 3_ | _его файлы_ |

## Готовый перевод

Собранный архив с переводом для игроков распространяется через **Releases**
репозитория, а не хранится в дереве исходников.
