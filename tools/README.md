# Инструменты переводчика (аддоны Translator++)

Вспомогательные аддоны для **[Translator++](https://dreamsavior.net/translator-plusplus/)**,
которыми велась работа над переводом. Каждый — самостоятельная папка аддона;
ставится копированием в `Translator++/www/addons/` (перезапустить Translator++ после).

> Игра: Demo **v0.49**.

## `linesChecker` — Lines Checker (проверка длины через игру)

Клиент-аддон: подсвечивает в гриде ячейки, чей перевод **не влезает** в игровое
окно диалога/описания. Меряет не эвристикой, а живым движком **RGSS3** — шлёт
текст по HTTP на `LinesCheckerServer` внутри игры (RMModLoader HTTP API,
`127.0.0.1:27420`) и красит ячейку:

- жёлтая — запрос в полёте (правка ячейки, debounce 300 мс);
- красная — движок сообщил, что текст не помещается;
- без цвета — ок.

Проверяет перевод (правый непустой столбец); опционально — и оригинал (col 0).
При открытии файла шлёт один batch-проход пачками по 300 строк.

**Зависимость:** в игре должен быть запущен `LinesCheckerServer` (компонент
RMModLoader), иначе мерить некому. `autoload: false` — включается вручную.

## `jsondb-choice-fix` — JsonDB Choice Guard Fix

Самовосстанавливающийся патч парсера `rmrgss`. Возвращает guard'ы
`isTranslatableProperties()` для команд **Choices (102)** и **Choice Options (402)**
в `JsonDB.fetchCommandList()`. Без них worker-поток `rmrgss` падает, когда
значение выбора — не переводимая строка, и это рушит весь batch-парс (и,
как следствие, экспорт).

Патч правит `rmrgss/JsonInterface/JsonDB.js` на диске **идемпотентно** и
переприменяется при каждом запуске — переживает обновления Translator++ / rmrgss,
которые перезаписывают ядро. `autoload: true`, требует Translator++ ≥ 7.1.1.
